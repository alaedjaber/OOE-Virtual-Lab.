# 👁️ Eye Tracker Simulation

## Physics & Optical Principles
This model simulates basic **Geometric Optics** involved in eye-tracking systems, focusing on light reflection off the cornea (Purkinje images) and gaze vector estimation.


## How to Use
1. Open `index.html` in any modern browser.
2. Move your mouse (or touch) to simulate eye movement.
3. Observe the calculated **Gaze Point** on the screen.
4. Use the visual rulers to estimate angles of incidence.


## Technical Details
- Real-time coordinate mapping using JavaScript.
- Responsive design for different screen sizes.
- Single-file architecture for easy deployment.
