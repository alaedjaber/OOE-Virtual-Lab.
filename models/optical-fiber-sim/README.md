## How to Use
1. Open `index.html` in any modern browser.
2. Adjust the **Refractive Indices** ($n_{core}$, $n_{cladding}$) using the sliders.
3. Change the **Incident Angle** to observe:
   - **Refraction**: When angle < Critical Angle.
   - **Total Internal Reflection**: When angle > Critical Angle (light stays trapped).
